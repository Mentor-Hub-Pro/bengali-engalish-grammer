## 🚀 Getting Started (Local Setup)

If you want to run this project locally on your machine, follow these simple steps:

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, etc.)
- [Git](https://git-scm.com/downloads) installed on your computer.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mentor-hub-pro/bengali-engalish-grammer.git
   ```
2. **Navigate to the project directory**
   ```bash
   cd bengali-engalish-grammer
   ```
3. **Run the application**
   
   Since this is a static website, you can simply open the `index.html` file in your browser. 
   
   *Alternatively, for a better development experience, use a live server:*
   - If you use VS Code, install the **Live Server** extension.
   - Right-click on `index.html` and select **"Open with Live Server"**.
